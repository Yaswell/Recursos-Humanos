Lenguaje de consulta estructurado (SQL)Structured Query Language (SQL)

SQL es un lenguaje que se usa en muchas bases de datos relacionales para administrar los datos en una base de datos.SQL is a language that's used in most relational databases for managing data in a database. Incluye los comandos que le permiten recuperar datos y actualizarlo y que le permiten crear, modificar y administrar datos en tablas de base de datos.It includes commands that let you retrieve data and update it, and that let you create, modify, and manage data in database tables. SQL no es un lenguaje de programación (por ejemplo, C#).SQL is different than a programming language (like C#). Con SQL, indicar la base de datos, lo que desea y trabajo de la base de datos consiste en averiguar cómo obtener los datos o realizar la tarea.With SQL, you tell the database what you want, and it's the database's job to figure out how to get the data or perform the task. Estos son algunos ejemplos de algunos comandos SQL y qué hacer:Here are examples of some SQL commands and what they do:

Select * From Movies

SELECT ID, Name, Price FROM Product WHERE Price > 10.00 ORDER BY Name

La primera Select instrucción obtiene todas las columnas (especificado por *) desde el películas tabla.The first Select statement gets all the columns (specified by *) from the Movies table.

El segundo Select instrucción recupera las columnas ID, Name y precio de los registros en el producto tabla cuyo valor de la columna precio es superior a 10.The second Select statement fetches the ID, Name, and Price columns from records in the Product table whose Price column value is more than 10. El comando devuelve los resultados en orden alfabético según los valores de la columna nombre.The command returns the results in alphabetical order based on the values of the Name column. Si no hay registros que coincidan con los criterios de precio, el comando devuelve un conjunto vacío.If no records match the price criteria, the command returns an empty set.

INSERT INTO Product (Name, Description, Price) VALUES ('Croissant', 'A flaky delight', 1.99)

Este comando inserta un nuevo registro en el producto tabla, al establecer la columna nombre para el precio, la columna Description para "Los usuarios no confiables A" y "Croissant" a 1.99.This command inserts a new record into the Product table, setting the Name column to "Croissant", the Description column to "A flaky delight", and the price to 1.99.

Tenga en cuenta que, al especificar un valor no numérico, el valor esté entre comillas simples (no comillas dobles, como en C#).Notice that when you're specifying a non-numeric value, the value is enclosed in single quotation marks (not double quotation marks, as in C#). Utilice estas comillas alrededor de los valores de texto o fecha, pero no en torno a los números.You use these quotation marks around text or date values, but not around numbers.

DELETE FROM Product WHERE ExpirationDate < '01/01/2008'

Este comando elimina los registros de la producto tabla cuya columna de fecha de expiración es anterior al 1 de enero de 2008.This command deletes records in the Product table whose expiration date column is earlier than January 1, 2008. (El comando supone que el producto tabla tiene una columna de este tipo, por supuesto.) Se especifica la fecha en formato MM/DD/AAAA, pero se debe escribir en el formato que se usa para la configuración regional.(The command assumes that the Product table has such a column, of course.) The date is entered here in MM/DD/YYYY format, but it should be entered in the format that's used for your locale.

El Insert y Delete comandos no devuelven conjuntos de resultados.The Insert and Delete commands don't return result sets. En su lugar, devuelven un número que indica el número de registros afectado por el comando.Instead, they return a number that tells you how many records were affected by the command.

Para algunas de estas operaciones (por ejemplo, insertar y eliminar registros), el proceso que solicita la operación debe tener los permisos adecuados en la base de datos.For some of these operations (like inserting and deleting records), the process that's requesting the operation has to have appropriate permissions in the database. Es por eso para bases de datos de producción a menudo tiene que proporcionar un nombre de usuario y una contraseña cuando se conecta a la base de datos.That's why for production databases you often have to supply a user name and password when you connect to the database.

Existen docenas de comandos SQL, pero siguen un patrón similar a los comandos que aparecen aquí.There are dozens of SQL commands, but they all follow a pattern like the commands you see here. Puede usar los comandos SQL para crear tablas de base de datos, contar el número de registros en una tabla, calcular precios y realizar muchas más operaciones.You can use SQL commands to create database tables, count the number of records in a table, calculate prices, and perform many more operations.
